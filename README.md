## 📆 진행 방식

## 📜 진행 방식
- 발표자는 매주 **수요일 자정까지 발표자료를 PR**로 올립니다.  
- 그 외 인원은 **목요일 21시까지** 각 장마다 질문 혹은 중요 포인트를 Issue로 등록합니다.  
- 발표가 끝나면 10분 내외 회고 진행  

## 🖥 발표

- 발표자는 발표자료 준비 및 발표 진행  
- 발표 중 해당 주에 올라온 Issue 리뷰 및 토론
